## 🚀 快速开始

### Android应用安装步骤
1. **下载APK文件**: 点击上方下载按钮
2. **启用未知来源**: 在Android设置中允许安装未知来源应用
3. **安装应用**: 点击APK文件进行安装
4. **网络配置**: 启动应用后配置WiFi连接参数
5. **开始监控**: 输入系统IP地址，开始实时监控

### Windows应用使用步骤
1. **下载EXE文件**: 点击上方下载按钮
2. **运行程序**: 双击EXE文件直接运行（无需安装）
3. **网络配置**: 在设置面板中配置网络连接
4. **系统连接**: 连接到植物监测系统
5. **开始使用**: 享受完整的监控体验

## 🔧 技术规格

- **开发框架**: Flutter 3.x

## 🐛 已知问题

### 当前版本已知限制
- Android应用需要手动启用未知来源安装
- Windows应用首次运行可能触发Windows Defender警告
- 大量数据场景下图表渲染可能有延迟
- 网络不稳定时可能出现连接中断

### 解决方案
- 正在申请Google Play上架，预计v2.1.0版本
- 正在申请代码签名证书
- 优化图表渲染性能
